Important Notes

Two important points to note when using handshake domains.

Don't Collect Sensitive Information

SSL is currently not supported by handshake so be mindful of what data you collect via the site if any. Directing users to 3rd party systems to make contact is recommended. Best Practice Examples: Click to email, Discord, Telegram, Send a Facebook Message.

Email Is Available Through Hmail

Visit Hmail.app to setup private email on any handshake domain. The current cost is 1 hns per month for private email using any handshake domain name you own.
